========== 🔧 セットアップ手順 ==========

1. プロジェクトをダウンロード
   git clone https://github.com/[your-username]/sedori-app.git

2. ディレクトリに移動
   cd sedori-app

3. 環境設定
   - QUICKSTART.md を読む
   - SETUP_GUIDE.md でFirebase設定
   - .env.example を .env.local にコピー

4. インストール
   npm install

5. 開発サーバー起動
   npm run dev

6. ブラウザで確認
   http://localhost:3000

========== 📊 主要な機能 ==========

✅ 商品管理
   - 商品名・購入価格・ポイントを登録
   - ステータス管理（待機中 → 売却済）

✅ 売却管理
   - 売却先・売却価格・売却日を入力
   - 利益・P利益を自動計算

✅ ダッシュボード
   - 総売上・総利益・利益率を表示
   - 在庫価格を把握

✅ データ操作
   - CSV出力（Google Sheets対応）
   - Firebase クラウド保存
   - 複数デバイス自動同期

========== 🚀 本番デプロイ ==========

Vercel (推奨・無料)
  → https://vercel.com

Netlify
  → https://netlify.com

セルフホスティング
  → Node.js + Express + PM2

========== 📚 技術スタック ==========

Frontend:
  - React 18 + TypeScript
  - Vite (高速ビルド)
  - Tailwind CSS (スタイリング)
  - Lucide React (アイコン)
  - Zustand (状態管理)
  - date-fns (日付操作)

Backend:
  - Firebase Authentication
  - Firestore Database
  - Firebase Storage

========== 🔒 セキュリティ ==========

- Firebase Authentication (ユーザー認証)
- Firestore Security Rules (データ保護)
- Environment Variables (.env.local)
- HTTPS通信 (本番)
